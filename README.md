# Aether-Core

Aether-Core: System Reclamation
Aether-Core is a high-octane 2D Tower Defense game built in Java using a custom engine architecture. Set in the dying world of Aetheria, players must protect the "Technological Heart" from the Shard Scourge—a relentless army of stone and scrap-metal hybrids determined to reclaim the world's energy.

Key Technical Features:

Waypoint-Based Pathfinding: An efficient custom navigation system that allows enemies to traverse complex, pre-defined terrain with pixel-perfect precision.

Modular Tower System: A deep strategic layer featuring 20 unique towers, each supporting a "Socket & Module" system. Players can craft and insert modules to dynamically alter tower stats (Range, Damage Type, Special Effects) during live gameplay.

JSON-Driven Wave Controller: A fully decoupled wave management system. Level designers can define enemy counts, spawn intervals, and tier-based scaling within a waves.json file, allowing for infinite level expansion without code changes.

Object-Oriented Design: Implements clean architectural patterns (Factory, Registry, and Singleton) to manage a diverse cast of 40+ enemy types and complex tower interactions.

Dynamic Resource Economy: Features a dual-currency system (Gold and Aether Shards) that balances immediate defense spending with long-term modular crafting.

The Mission:
Defend the Heart. Upgrade your Paladin. Prevent System Reclamation.
